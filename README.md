# Kivy-Mobile-Calculator

This project is to practice skills for mobile devvelopment using my main language: Python, and the kivy library.

The project consist on making multiple types of calculators into de app that an universitary student would use.

Averager: This calculator to check the final grade by the obtained by calculate grades and their percent. (WIP)

Mathematics: Standard calculator, it can do standard mathematics operations. (WIP)

Scientific: Calculator that is made to calculate physics related calculations, and maybe, do chemistry related calculations. (WIP)
